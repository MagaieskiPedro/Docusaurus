# METODOS HTTP: GET,PUT,DELETE
### Descrição dos campos:

cod é o código da turma

turrma é o nome da turma

### Exemplo de JSON de entrada
```json    
{
    "cod":"12",
    "turrma":"2DS-TB"
}
```   
### Exemplo de JSON de saída
```json    
{
	"id": 1,
	"cod": "12",
	"turrma": "2DS-TB"
}
```
